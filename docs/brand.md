# Chrome Relay Brand Notes

## Direction

Chrome Relay should feel like a local, trustworthy control bridge rather than a generic automation bot. The visual language should be precise, slightly hardware-ish, and calm: dark blue-green, warm cream, electric cyan, and a small yellow signal accent.

## Logo Ideas

1. **Relay Bolt**: a lightning-shaped bridge between two rounded browser panels. This is the current implemented icon. It reads well at small sizes and avoids Chrome-logo lookalikes.
2. **Local Loop**: two small nodes connected by a loop around a terminal cursor. Stronger developer feel, but slightly less obvious as browser control.
3. **Glass Port**: a browser window with a glowing local bridge socket in the corner. Good for landing-page illustration, weaker as a 16px icon.

## Chosen Icon

Use **Relay Bolt** for the first marketplace submission.

Reasons:

- Distinct from Google Chrome branding.
- Communicates fast local handoff.
- Works on light and dark backgrounds.
- Can scale from 16px extension icon to 440x280 promo art.
